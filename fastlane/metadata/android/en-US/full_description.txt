Prayer Times helps you keep track of the five daily prayers with a clear home screen, optional reminders, a full-month planner, and a Qibla compass.

Key features:
- Prayer times for today with quick, readable layout
- Optional notifications for the five daily prayers
- Monthly overview for planning ahead
- Qibla direction using device sensors
- GPS location or manual coordinates
- Place search where supported
- Multiple calculation sources
- English, Swedish, and Arabic (RTL)
- Light, dark, and system theme options
- Android home-screen widget

The app is open source under Apache-2.0.
